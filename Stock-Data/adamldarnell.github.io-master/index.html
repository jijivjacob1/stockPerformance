WeatherPy
    The core goal of this project was to analyze how weather patterns differ the closer one gets to the equator. The data was gathered from 500 different cities using the OpenWeatherMap API. After the data was gathered, Matplotlib was used to visualize various comparisons. These visualizations can be viewed in more detail using the links below.
